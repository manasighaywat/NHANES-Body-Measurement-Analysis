# NHANES Body Measurement Statistical Analysis

## 📌 Project Overview
Statistical analysis of adult male and female body measurements
from the NHANES 2020 dataset using Python.

## 🛠️ Tools & Libraries
- Python
- NumPy
- Matplotlib
- Seaborn

## 📈 Visualizations Created
- Histograms of body measurements
- Box-and-whisker plots (male vs female comparison)
- Pairplot scatterplot matrix

## 📁 Dataset
- Source: NHANES 2020 (CDC)
- Files: nhanes_adult_female_bmx_2020, nhanes_adult_male_bmx_2020

## 🚀 How to Run
1. Clone this repository
2. Install libraries: pip install numpy matplotlib seaborn
3. Open Capstone_project_1_nhanes_data_analysis.ipynb in Jupyter Notebook
4. Run all cells
